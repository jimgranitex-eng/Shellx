# ShellX Architecture

ShellX is composed of three layers:

## 1. ShellX Core (V1)
Simple report generator.

## 2. ShellX Engine (V2–V500)
Multi-agent pipelines:
- deep audit
- debug
- architect
- memory
- safe-mode autofix
- verify

## 3. ShellX V3 Cognitive Layer
Collector → Merger → Cognitive Compressor → Summary → Full Report → LinkX → Stone
