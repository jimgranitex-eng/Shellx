# ShellX — Command Examples

This guide provides real-world examples of how to use ShellX across V1, V2, V3, Pro, and Superman modes.

---

# 🧱 Core Examples (V1)

## Basic Report
```
shellx "Fix the shader pipeline and update the renderer."
```

## Preflight Check
```
shellx preflight
```

## View Last Report
```
shellx look
```

---

# 🧩 Worker Examples (V2)

## Deep Audit
```
shellx deep-audit
```

## Architecture Analysis
```
shellx architect
```

## Safe Autofix
```
shellx autofix --safe
```

# 🦸 Superman Examples

shellx-superman "audit everything and stabilize the engine"

shellx-superman "map architecture and check memory integrity"
