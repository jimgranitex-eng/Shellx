# ShellX V4 Roadmap

ShellX V4 will transform ShellX from an embedded KickerOS subsystem into a fully modular standalone engine.

## Goals
- Plugin system for workers
- Unified worker interface
- Strict command parser
- Cloud-based LinkX timeline
- Stone diff viewer UI
- VS Code extension with interactive panels
- ShellX Pro (commercial) support

## Architecture Changes
- Decouple ShellX from KickerOS
- Move pipelines into modules
- Add plugin loader
- Add worker registry
