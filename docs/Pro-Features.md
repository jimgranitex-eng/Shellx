## Superman Pro

The most powerful diagnostic mode in ShellX:

- **Deep architecture mapping** — full codebase structure analysis
- **Memory graph analysis** — track references, dependencies, and data flow
- **Worker orchestration** — coordinate all agents for maximum coverage
- **Autofix+** — safe mode and aggressive mode with rollback support
- **Multi-threaded verification** — parallel checks for speed and accuracy

```bash
shellx-superman-pro "full diagnostic and stabilize everything"
```

## Enterprise LinkX Timeline

- **Multi-project memory graph** — track evolution across repos and branches
- **Diff visualization** — see what changed between any two points in time
- **Time-travel debugging** — step back to any previous state
- **Cross-branch intelligence** — compare states across branches and teams

## Stone Diff Viewer

- **Visual snapshot comparison** — see what changed between stones
- **Restore snapshots** — revert to any previous stone
- **Generate recovery plans** — automated suggestions for reverting changes
- **Annotation support** — add notes to stones for future reference
