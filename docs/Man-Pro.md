# SHELLX-PRO(1) — User Commands

## NAME
shellx-pro — ShellX Pro commercial commands

## SYNOPSIS
```
shellx activate <license-key>
shellx-superman-pro "<message>"
shellx <pro-worker> [options]
```

## DESCRIPTION
ShellX Pro unlocks advanced diagnostics, workers, and enterprise features.

## COMMANDS

### `activate <license-key>`
Activates ShellX Pro.

### `shellx-superman-pro "<message>"`
Runs the full Pro diagnostic engine.

### `architect-pro`
Advanced architecture mapping.

### `memory-pro`
Enterprise memory graph analysis.

### `deep-audit-pro`
Enhanced deep audit.

### `autofix-pro`
Autofix+ (safe + aggressive modes).

### `verify-pro`
Pro-level verification.

### `timeline-pro`
Enterprise LinkX timeline tools.
