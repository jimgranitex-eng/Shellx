# ShellX Bridge Modules (Experimental)

These modules provide optional HTTP and browser automation bridges.

**Status:** Experimental — APIs may change. Not required for core ShellX CLI functionality.

| Module | Status | Description |
|--------|--------|-------------|
| grok_http_bridge.py | Experimental | Local HTTP helper on port 8765 |
| browser_bridge.py | Experimental | Browser automation bridge |
| browser_runner.py | Experimental | Browser script runner |
| headless_browser.py | Experimental | Headless browser controller |

To use these features, install additional Python dependencies:
```
pip install -r ../requirements.txt
```
