Commands Cheat Sheet

Complete reference for all ShellX commands across V1, V2, V3, and Superman.

🧠 Cognitive Mode (V3)

The flagship natural language interface. Describe what you want, get results.

Command Description Output
shellx --xx "<message>" Cognitive mode with full pipeline 1-2-3 summary + report
$ shellx --xx "Optimize database queries in the auth module"

🦸 Superman Mode

All-systems diagnostic engine. Runs the complete pipeline.

Command Description Pipeline
shellx-superman "<message>" Full diagnostic suite 9 workers

Superman runs:

  1. Smart input processing
  2. Deep audit
  3. Debug analysis
  4. Architect review
  5. Memory update
  6. Safe autofix
  7. Verification
  8. Stone creation
  9. LinkX update

🧱 Core Commands (V1)

Original ShellX command set.

Command Description
shellx "<message>" Legacy direct mode
shellx report Generate comprehensive report
shellx preflight System readiness check
shellx collect Collect system diagnostics
shellx look Inspect current state

🧩 Worker Commands (V2)

Individual worker access for specific tasks.

Command Worker Purpose
shellx deep-audit Auditor System analysis & bottleneck detection
shellx debug Debugger Issue identification & tracing
shellx architect Architect Structural recommendations
shellx memory Memory State management & LinkX sync
shellx autofix --safe Autofix Safe automatic repairs
shellx verify Verifier Change validation

🧬 Multi-Project Mode

Manage multiple repositories simultaneously.

Command Description
shellx multi-project "<instructions>" Run across multiple project contexts

🪨 Stone Commands

Recovery snapshot management.

Command Description
shellx stone list List all stones
shellx stone show <id> View stone details
shellx stone diff <id> View changes in stone
shellx stone restore <id> Rollback to stone state

🧠 LinkX Commands

Memory timeline and search.

Command Description
shellx linkx timeline View chronological history
shellx linkx show <id> View specific entry
shellx linkx search "<query>" Search history

🧪 Experimental (V4 Roadmap)

Coming in version 4.0.

Command Description Status
shellx plugin install <name> Install plugin Planned
shellx plugin list List installed plugins Planned
shellx engine status View engine health Planned

✔ Command Summary

ShellX commands fall into these categories:

  • Cognitive (V3) — Natural language interface
  • Diagnostic (V2) — Individual worker control
  • Core (V1) — Legacy commands
  • Superman — All-in-one diagnostic
  • LinkX — Memory & timeline
  • Stones — Recovery & rollback
  • Multi-Project — Cross-repo operations
  • V4 Experimental — Plugin system

💡 Pro Tip

Use shellx --xx for 90% of tasks. It's the most powerful command and handles intent parsing, worker orchestration, and summary generation automatically.
