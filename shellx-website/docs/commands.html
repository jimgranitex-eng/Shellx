ShellX CLI Commands

Install the current CLI package and use the supported ShellX commands.

Install

npm install -g shellx-dev-cli

Command Set

Command Description
shellx init Initialize ShellX project folders and baseline state.
shellx linkx init Initialize LinkX memory for the project.
shellx linkx scan Scan the project and update LinkX memory.
shellx linkx show Display current LinkX memory summary.
shellx linkx timeline Show LinkX timeline history.
shellx verify Verify ShellX and project state integrity.
shellx report Generate a report (use --format for detailed/full).
shellx --xx "<intent>" Run cognitive mode using natural-language intent.

Example Flow

shellx init
shellx linkx init
shellx linkx scan
shellx linkx show
shellx linkx timeline
shellx verify
shellx report
shellx --xx "stabilize the rendering pipeline"
