Commands Cheat Sheet

Current ShellX CLI command reference.

📦 Install

$ npm install -g shellx-dev-cli

⚙️ Core Commands

Command Description
shellx init Initialize ShellX and create .linkx/.reports/.stones
shellx verify Verify local ShellX project readiness
shellx report Generate summary diagnostics report
shellx --xx "<intent>" Run cognitive mode with natural-language intent

🧠 LinkX Commands

Command Description
shellx linkx init Initialize LinkX memory state
shellx linkx scan Scan project and update LinkX tracking data
shellx linkx show Display current LinkX memory state
shellx linkx timeline Show LinkX intent and stone timeline

💡 Pro Tip

Start with shellx init, then use shellx --xx for intent-driven work and shellx linkx timeline to review project history.
