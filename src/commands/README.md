# ShellX Commands

This directory contains all CLI command implementations for ShellX.

## Available Commands

### `shellx init`
Initialize ShellX and LinkX in your project.
Creates `.linkx/`, `.reports/`, and `.stones/` directories.

### `shellx linkx init`
Initialize LinkX memory system.

### `shellx linkx scan`
Scan your project and update LinkX memory.

### `shellx linkx show`
Display current LinkX memory state.

### `shellx linkx timeline`
Show LinkX timeline of all changes and intent history.

### `shellx verify`
Verify ShellX installation and configuration.

### `shellx report`
Generate diagnostic reports.
Options: `--format summary|detailed|full`

### `shellx --xx <intent>`
Cognitive mode: process natural language intent and run full pipeline.
